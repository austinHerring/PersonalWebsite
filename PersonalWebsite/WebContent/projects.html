Political Bill Classification

August, 2016

This project was done in conjunction with "2" other students (really 1). Many people don't have a clue on what congressional bills are currently being looked at or passed. The project seeks to solve that problem by finding bills relevant to a user based on what he likes. First we scraped the Bills from 1989-present day in the form of text files. Then we generated a Term-Document Matrix using a weighting function rooted in Bayesian probability. Next we used hierarchical support vector machines on the different vectors of the matrix as a way of feature extraction. Lastly, we used Latent Semantic Indexing and Term Frequency-Inverse Document Frequency to create weighted feature vectors that we then ran the hierarchy classifier to gain improved results.features.

Click to view the report with our conclusions and procedures. It is in a repository so the source code is within the same project directory.

DE2Bot Traveling Salesman Problem Algorithm

November, 2015

This project was done in collaberation with three other students. We wrote our implementation of the travelling salesman problem combining VHDL as our hardware desrciption language and Nios II Assembly language. The DE2 Simple Computer was made mobile by attaching it to a bot. Using a C++ preprocessing algorithm to order a given set of points we were able to hit at least twelve points in 12ft by 8ft grid in under a minute.
